- Lucide React + React Icons

## Desenvolvimento local

```bash
npm install
npm run dev
```

Abra [http://localhost:5173](http://localhost:5173)

## Simular o site do GitHub Pages localmente

```bash
npm run preview:pages
```

Abra [http://localhost:4173/portifolio/](http://localhost:4173/portifolio/)

## Hospedar no GitHub Pages

Tudo já está configurado no repositório. Siga estes passos **uma única vez**:

### 1. Ativar o GitHub Pages

1. Acesse [github.com/jeffSv16/portifolio/settings/pages](https://github.com/jeffSv16/portifolio/settings/pages)
2. Em **Build and deployment → Source**, selecione **GitHub Actions**
3. Salve (não precisa escolher branch nem pasta)

### 2. Publicar o site

Qualquer `push` na branch `main` dispara o deploy automaticamente.

Para publicar manualmente:

1. Vá em **Actions** no repositório
2. Selecione **Deploy to GitHub Pages**
3. Clique em **Run workflow**

### 3. Acessar o site

Após o workflow concluir (1–3 min):

**https://jeffsv16.github.io/portifolio/**
