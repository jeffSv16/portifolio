## Como visualizar

Abra o arquivo `index.html` no navegador ou use um servidor local:

```bash
npx serve .
```

## Deploy (GitHub Pages)

1. Vá em **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / pasta **/ (root)**
4. Salve — o site ficará em `https://jeffsv16.github.io/portifolio/`
